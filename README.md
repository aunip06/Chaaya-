🌿 Chaaya — Urban Shade Intelligence

Chaaya is an AI-powered urban shade intelligence platform designed for the MIT-WPU Kothrud campus in Pune. It provides real-time shade mapping, solar radiation insights, and heat risk analysis to help pedestrians navigate cooler, safer routes across campus.

🚀 Features
🌤 Real-Time Shade Mapping

An interactive campus map displaying:

Heat intensity zones (heatmaps)
Tree canopy coverage
Dynamic 3D building shadows
🤖 Chaaya AI Chatbot

A dedicated AI assistant that helps users with:

Finding optimal shaded routes
Identifying cooler zones on campus
Understanding plantation and greening priorities
📊 Environmental Metrics

Live monitoring of key outdoor conditions:

Heat Index – Combined effect of temperature and humidity
Tree Coverage – Percentage of green cover
Comfort Score (1–10) – Outdoor pedestrian comfort rating
UV Index & Wind Speed – Real-time environmental telemetry
☀️ Solar Radiation Analysis

Zone-wise comparison of solar intensity (e.g., Main Building vs. Sports Ground) to assess heat exposure patterns.

🌱 Plantation Insights

AI-based recommendations that highlight areas requiring additional tree plantation to improve thermal comfort and sustainability.

🛠 Tech Stack
Frontend: HTML5, CSS3 (custom properties), JavaScript (ES6+)
Mapping Library: Leaflet.js
Heatmap Visualization: leaflet-heat.js
Typography: Space Mono, DM Sans, Syne (Google Fonts)
Inference Engine: Hybrid rule-based + ML-ready model (v0.3)
📊 Model Details (v0.3)
Inputs:
Satellite imagery
NDVI (Normalized Difference Vegetation Index)
Solar position data
Building height data
Sensor telemetry
Accuracy: ~87.4% validated
Refresh Rate:
Sensor updates every 15 minutes
Shade maps updated every 6 hours
📍 Coverage

Currently, Chaaya supports:

MIT-WPU Kothrud Campus
Surrounding region within a 2 km radius
🎯 Purpose

Chaaya is built to:

Reduce pedestrian exposure to urban heat
Enable climate-aware navigation
Support data-driven urban greening strategies
